# note-architect

小説執筆補助ツールの実装ベースです。  
執筆・キャラ・世界観・年表に加えて、編集/削除やバックアップも可能です。

## セットアップ

```bash
npm install
npm start
```

`http://localhost:3000` へアクセスしてください。

## 実装済み機能

- シーン管理（作成 / 編集 / 削除）
- キャラクター管理（作成 / 編集 / 削除）
- 世界観エントリ管理（作成 / 編集 / 削除）
- 年表イベント管理（作成 / 編集 / 削除）
- localStorage 永続化
- JSON エクスポート / インポート
- 全データ削除
- ヘルスチェック API: `GET /api/health`
