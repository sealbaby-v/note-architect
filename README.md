# note-architect

モダンUIの長編小説執筆補助ツールです。ローカルで起動し、章・シーン・キャラクター・世界観・年表・AI補助を実際に操作できます。
小説執筆補助ツールの実装ベースです。

## セットアップ

```bash
npm install
npm start
```

## 主な機能

- ダッシュボード（章数/シーン数/文字数など統計表示）
- 章・シーン管理（追加 / 編集 / 並び替え / 削除）
- キャラクター管理（追加 / 編集 / 並び替え / 削除）
- 世界観管理（追加 / 編集 / 並び替え / 削除）
- 年表管理（追加 / 編集 / 並び替え / 削除）
- AI補助（プロット提案 / リライト / 批評）
- JSONエクスポート / インポート
- localStorage 永続化
- `GET /api/health`

## 注意

- AIは現状スタブ実装です（実LLM未接続）。
## 実装済み

- 章管理（追加・並び替え・削除）
- シーン管理（追加・章紐付け・並び替え・削除）

- AI補助API（プロット提案・リライト・批評）
